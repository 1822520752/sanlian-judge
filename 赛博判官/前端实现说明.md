# 赛博判官 — 前端实现说明

> 版本：v1.0 · 2026-07-16

## 1. 总体架构

- **单 HTML + 多 JS 模块** 的移动端优先 SPA
- 通过 CDN 引入 `tailwindcss` 与 `html2canvas`，**不依赖任何 npm 包**
- 4 个 section 容器：`#page-home` / `#page-loading` / `#page-report` / `#modal-rank`
- 全局命名空间挂到 `window.CyberJudge*` 系列

### 文件清单

| 路径 | 作用 |
|---|---|
| `index.html` | SPA 容器，4 个 section 切页 |
| `static/css/style.css` | 自定义动画 + 样式（Tailwind 补充） |
| `static/js/cache.js` | localStorage 缓存（TTL 24h + LRU 10） |
| `static/js/app.js` | state + 首页 + 加载页 + 通用工具 |
| `static/js/report.js` | 报告页 8 模块渲染 |
| `static/js/share.js` | 1080x1920 分享卡（html2canvas） |
| `static/js/rank.js` | 排行榜 Modal |

加载顺序（HTML 中按依赖排列）：
```
cache.js → app.js → report.js → share.js → rank.js
```
其中 `app.js` 通过 `window.CyberJudge` 暴露 state / fetch 等；`report.js` 会在 DOM ready 后由 `app.js` 调用 `render(state)`。

---

## 2. 状态管理

`window.CyberJudge.state`（app.js 内）：

```js
{
  currentPage: 'home' | 'loading' | 'report',
  uid: '',
  profile: null,    // /api/profile data
  report: null,     // /api/analyze data
  ranking: null,    // /api/rank data
  fromCache: false,
}
```

切页通过 `CyberJudge.showPage(name)`：隐藏其它 section + 触发 fadeIn 动画 + 滚到顶。

## 3. API 调用约定

| API | 方法 | 超时 | 失败行为 |
|---|---|---|---|
| `/api/profile?uid={uid}` | GET | 10s | toast 错误 → 回首页 |
| `/api/analyze` (POST) | POST {uid, profile} | 12s | toast 错误 → 回首页 |
| `/api/rank?type=craziness&page=1&limit=20` | GET | 默认 | 弹窗内显示"加载失败" |

**15 秒兜底**：loading 页面停留超过 15 秒，强制 toast 提示并返回首页。

**进度文字顺序**：
1. `正在连接B站...`（profile 调用期间）
2. `正在召唤AI判官...`（analyze 调用期间）
3. `正在书写前世...`（拿到结果后 350ms 跳报告页）

## 4. 首页

- Logo 浮动 + 标题渐变色（`#FB7299 → #FF6B81`）
- 输入框：56px 高 / 28px 圆角 / 18px 字号 / 聚焦粉色 + 阴影
- 主按钮：渐变粉底 + 阴影 0 4px 16px rgba(251,114,153,0.4)
- 示例 UID：白底 + 1px 边框 + 悬停浅粉
- 验证：非数字 → 红色边框 + 抖动 + 提示「UID只能是数字」

## 5. 报告页 8 模块

| # | 模块 | 数据来源 | 关键动画 |
|---|---|---|---|
| 1 | 核心身份卡 | profile | 头像 360° 旋转 + 数字 count-up |
| 2 | 弹幕人格 | report.personaType | 类型标签 bounceIn + 星级填充 |
| 3 | 赛博前世 | report.pastLife | 大 emoji dropIn + 引用卡片 |
| 4 | 精神状态 | report.mentalState | 仪表盘进度条 grow + 锚点高亮 |
| 5 | 2026 运势 | report.fortune2026 | 4 卡 stagger 弹入 |
| 6 | 灵魂伴侣 | report.soulMate | 头像左/右滑入 + 爱心浮动 |
| 7 | 弹幕风格 | report.danmuStyle | 标签从中心扩散 + 徽章 popOut |
| 8 | 离谱指数 | report.craziness | 进度条 grow + 数字 count-up |

入场动画统一走 IntersectionObserver，触发后追加 `in-view` 类；CSS 中对应关键帧 `slideUp 600ms cubic-bezier(0.16, 1, 0.3, 1)`。

`CyberJudgeReveal` 在 `app.js` 暴露 `observeAll()`，每次 `renderReport` 都会重新注册新模块元素。

## 6. 分享卡

- 容器：`#share-target`（隐藏于 `left: -10000px`）
- 尺寸：1080×1920
- 内容三段：身份卡 + 弹幕人格 + 离谱指数
- 头像 `crossorigin="anonymous"` + 失败回退
- `html2canvas({ useCORS: true, allowTaint: false, scale: 1 })`
- 自动下载 `cyber-judge-{uid}.png`
- 失败 / 跨域被拦截时给出友好中文提示

---

## 7. 排行榜 Modal

- 入口：报告页导航栏 `🏆` 按钮
- 打开 / 关闭：fadeIn + slideUp（移动端从底部弹出，桌面端居中）
- 关闭：点遮罩 / 点 × / 按 ESC
- 30 秒内不重复拉取
- 失败：弹窗内"排行榜加载失败"提示，不影响报告页

## 8. 缓存

`window.CyberJudgeCache`：

| API | 行为 |
|---|---|
| `get(uid)` | 命中且 < 24h 返回数据；过期或不存在返回 null |
| `set(uid, data)` | 写入 `{ts, data}`，更新 LRU 索引 |
| `size()` | 当前缓存 UID 数 |
| `clear()` | 清空全部 |

- 键名：`cyber-judge:cache:{uid}`
- 索引：`cyber-judge:cache-index`（数组，记录 `{uid, ts}`）
- LRU 上限 10 个 UID，超出按 `ts` 升序淘汰
- 命中后报告页顶部出现黄色 banner："⚡ 已展示缓存结果（24小时内）"

## 9. 响应式

- `body` 包裹 `.max-w-app`（640px）容器，移动端 100% 宽
- 触摸目标 > 44px（按钮高 48–52px）
- 字体最小 14px
- 减少动画 (`@media (prefers-reduced-motion: reduce)`)
- 移动端断点 `< 380px` 缩小部分字号

## 10. 错误处理

| 场景 | 提示 |
|---|---|
| UID 为空 | "请输入B站UID" |
| UID 含非数字 | "UID只能是数字"（自动过滤 + 抖动） |
| UID 长度异常 | "UID长度应在1-12位" |
| profile 超时 | "B站数据超时（10秒）" |
| analyze 超时 | "AI分析超时（12秒）" |
| profile 404 | 使用后端 `error` 字段（如"用户不存在"） |
| 整体超时 | 15s 兜底 toast "网络有点慢，请稍后重试" |
| 分享截图失败 | "截图失败：{message}" |

所有错误通过 `showToast(msg, 'error', 2400)` 弹出，**不暴露堆栈**。

---

## 11. 已知边界 / 后续工作

1. **B站头像 CORS**：B站 CDN 默认允许 CORS，分享卡可正常截图；若将来关闭 CORS，需要后端代理下载头像。
2. **浏览器兼容**：依赖 `IntersectionObserver` / `requestAnimationFrame` / `localStorage` / `fetch` / `Promise` / html2canvas。`prefers-reduced-motion` 已处理。
3. **排行榜**：依赖后端 `/api/rank` 实现；本前端已做 30s 内存级缓存。
4. **缓存容量**：超过 10 UID 时按时间淘汰最旧；配额满时尝试 `setItem` 失败则静默忽略。
5. **Tailwind CDN 警告**：生产环境会输出 `cdn.tailwindcss.com should not be used in production` 控制台告警，但不影响功能。
