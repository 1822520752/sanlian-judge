### 2.2 组件细节

**输入框:**
```
- 高度: 56px
- 圆角: 28px (全圆角)
- 边框: 2px solid #E0E0E0 (聚焦: #FB7299)
- 内边距: 0 24px
- 字体: 18px
- 阴影: 0 2px 8px rgba(0,0,0,0.06)

验证状态:
- 空值 → 粉色边框+抖动动画
- 数字 → 正常
- 非法字符 → 红色边框+提示
```

**主按钮:**
```
- 背景: linear-gradient(135deg, #FB7299, #FF6B81)
- 文字: 白色, 18px, Bold
- 高度: 52px
- 圆角: 26px
- 宽度: 100%
- 阴影: 0 4px 16px rgba(251,114,153,0.4)
- 悬停: transform: translateY(-2px), 阴影增强
- 禁用: opacity: 0.5, cursor: not-allowed
```

**示例UID标签:**

## 4. 动画规范

```
入场动画 (每个模块):
  1. 模块从下方滑入 (translateY: 40px → 0)
  2. 透明度从0到1
  3. 持续时间: 600ms
  4. 缓动函数: cubic-bezier(0.16, 1, 0.3, 1) (easeOutExpo)
  5. 使用 IntersectionObserver 触发

特殊动画:
  - 头像旋转: 360° + 弹性
  - 星级填充: 从0到目标星级
  - 进度条: 从0%到目标百分比
  - 数字跳动: 目标数字的递增动画 (requestAnimationFrame)
  - 标签扩散: 从中心向外弹射

加载动画:
  - 脉冲圆点 (3个点交替缩放)
  - 进度文字: "正在连接B站..." → "正在召唤AI判官..." → "正在书写前世..."
```

## 6. 响应式断点

```
移动端:  < 640px    (主目标, 100%宽度)
平板:    640-1024px  (居中640px卡片)
桌面:    > 1024px    (居中640px卡片+装饰背景)

重点优化移动端:
  - 触摸目标 > 44px
  - 字体不小于14px
  - 避免hover-only交互
  - 滑动流畅 (60fps)
```

## 7. 字体加载

```
方案: 系统字体栈 (零额外请求)

font-family: -apple-system, BlinkMacSystemFont,
             "Segoe UI", "PingFang SC",
             "Hiragino Sans GB", "Microsoft YaHei",
             sans-serif;

理由:
  - B站用户大部分在中国，系统字体加载最快
  - PingFang SC (iOS) + Microsoft YaHei (Windows) 质量高
  - 零网络请求，页面秒开
```
